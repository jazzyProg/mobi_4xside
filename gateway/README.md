# Production System Gateway

Единая точка входа для всех микросервисов системы контроля качества.

## Архитектура

                  ┌─────────────────┐
                  │  NGINX Gateway  │
                  │   Port: 8080    │
                  └────────┬────────┘
                           │
     ┌─────────────────────┼──────────────────────┐
     │                     │                      │
┌────▼────┐          ┌─────▼─────┐         ┌─────▼─────┐
│   QC    │          │  Camera   │         │  Signals  │
│ Service │          │    API    │         │    API    │
│ :8001   │          │   :8003   │         │   :8002   │
└────┬────┘          └─────┬─────┘         └─────┬─────┘
     │                     │                     │

                    │    :8000    │

## Сервисы

| Сервис | Порт | Путь API | Описание |
|--------|------|----------|----------|
| Gateway | 8080 | `/health` | Единая точка входа |
| QC Service | 8001 | `/api/qc/` | Контроль качества (YOLO) |
| Camera API | 8003 | `/api/camera/` | Управление камерой |
| Signals API | 8002 | `/api/signals/` | Управление лампами |
| Products API | 8000 | `/api/products/` | Работа с БД |
